Références/consignel-js

Le script utilisateur consignel.js envoie les demandes des utilisateurs au serveur et affiche les réponses

  1. Le script anime la page web du ↺onsignel en sélectionnant guidant et en informant l'utilisateur au fur et à mesure
  2. Le script permet le passage entre les différentes parties utilisation, inscription, confirmation et préférences
  3. Le script code les envoi vers le serveur et décode les retours du serveur
  4. Rôle des différentes parties accessibles par le menu général
    • Insc: Inscription code et mode de passe pour accéder au contenu privé
    • Util: Dire ce qu'on offre, ce qu'on demande et montrer le code QR de notre proposition
    • Conf: Accepter, négocier, refuser la proposition d'un autre ou renouveler, annuler et voir ses propres propostions
    • Préf: Ajuster l'application selon ses propres préférences et suivre son fonctionnement
