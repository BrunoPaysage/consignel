Références/consignel-php

Le script serveur consignel.php identifie les utilisateur et renvoi les réponses à leurs demandes

  1. Le script ouvre une session de durée limitée en acceptant uniquement les utilisateurs connus
  2. Si l'utilisateur est reconnu dans une session ouverte, le script traite sa demande en fonction des autorisations
  3. Le script renvoie une chaîne par question
  4. Tous les transferts et les stockages passent par un nettoyage et un cryptage.
  5. Si l'utilisateur est correctement identifié, la session est ouverte, le serveur retourne un code de session et le résumé de compte de l'utilisateur
  6. Si la session est déjà ouverte, le serveur renvoie une chaîne.

Éléments traités par le script

  • Les variables de transfert en provenance de l'utilisateur
  • Les demandes autorisées
  • Les identifiants et les réponses aux demandes
