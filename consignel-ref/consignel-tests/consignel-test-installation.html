Chargement et ouverture
  1. Cloner le Git ou chargez le zip du projet ↺onsignél sur votre serveur html php
  2. Aller sur votre serveur avec un navigateur
  3. Charger la page index.html à la racine du dossier que vous venez d'installer. Typiquement à partir du zip http://localhost:8888/consignel-master/index.html
  4. Tests complémentaires
    • Essayez de charger la page avec un serveur html sans php
    • Essayez sans javascript

Test des propositions
  1. Rechargez la page. Le point d'insertion clignote dans la barre de recherche
  2. Tapez une lettre dans la barre de recherche d'articles
  3. Cliquez sur je demande
  4. Tapez une lettre dans la barre de recherche d'articles
  5. Ajoutez d'autres items
  6. Modifiez les items
  7. Changez les items d'un bord à l'autre entre l'offre et la demande
  8. Supprimez les items en cliquant sur le titre puis sur la poubelle ou en les glissant sur le bord gris

Test de confirmation d'une proposition
  1. Relancez la page
  2. Connectez l'utilisateur autre sa réserve est de 250 ↺
  3. Tapez la lettre a dans la barre de recherche d'article et cliquez sur "produit de mon jardin"
  4. Changez la quantité à 10 kg la réserve passe à 229,5 ↺
  5. Cliquez sur "Je demande"
  6. Tapez la lettre a dans la barre de recherche d'article et cliquez sur "produit de mon jardin"
  7. Le code QR apparait en pâle
  8. Cliquez sur le code QR pour confirmer la proposition
