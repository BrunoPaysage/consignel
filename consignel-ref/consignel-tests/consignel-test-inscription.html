Test de l'inscription d'un nouvel utilisateur

Tests à effectuer

Inscription nouvel utilisateur
  1. Inscrire dans l'offre : vérification d'utilisateur unique 1h
  2. Inscrire dans la demande : inscription d'utilisateur unique 1h
  3. Cliquer sur le code QR
  4. Connectez l'utilisateur "autre" avec son mot de passe
  5. Recliquez sur le code QR. Le message indiquant qu'il manque A qui doit apparaître.
  6. Indiquez le nom public d'un utilisateur déjà inscrit dans "A qui ?". Le message de nom public déjà utilisé doit apparaître.
  7. Indiquez "autre2" dans "A qui ?". Un nouvel utilisateur inscription a été ajouté dans .baseconsignel3
  8. Scannez le code QR
  9. Inscrivez "autre2" comme identifiant d'utilisateur
  10. Inscrivez "inscription" dans Secret
  11. Inscrivez "autre2identifiant" dans l'identifiant de compte (confirmer avec "autre2identifiant")
  12. Inscrivez "autre2secret" dans le mot de passe (confirmez avec "autre2secret"). Fermez la fenêtre de confirmation que l'identifiant et le mot de passe on été chagés
  15. Tests complémentaires
