Test de première installation et vérifications sans configuration particulière

Tests à effectuer

  1. Faire une offre et une demande. Le code apparaît en pâle. La proposition n'est pas confirmée
  2. Clic sur le code QR pour confirmer la proposition. On passe par la procédure d'identification si c n'est pas déjà fait. Il faut alors recliquer sur le code QR pour confirmer la proposition.
  3. Scanner le code QR
  4. Connecter un autre utilisateur que celui qui a fait la proposition. Il doit s'identifier. La proposition doit être présentée du point de vue de l'utilisateur qui a scanné le code QR.
  5. Tests complémentaires
    • Essayer avec une demande à qui
    • Essayer sans connexion réseau
