.baseconsignel0 : Identifiants de session de l'utilisateur protégé par .htaccess

Références/consignel-base/0/contrôle session

Le script php utilise cette base pour autoriser les sessions, vérifier les sessions en cours et le dépassement de temps de connexion. Le script php fait aussi le nettoyage de la liste des sessions autorisées lorsque le temps est dépassé.
