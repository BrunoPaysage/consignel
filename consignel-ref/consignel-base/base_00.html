consignel-base/00/ : Base des données personnelles des utilisateurs.

Références/consignel-base/00/base personnelle

Le script php utilise cette base pour mettre à jour les données des utilisateurs

Cette base est organisée par identifiant des utilisateurs. 00/00/00/0/utilisateur-fichiers.json
Répertoires et sous répertoires codés à partir de l'identifiant codé de l'utilisateur

Le répertoire contient les fichiers permettant le suivi des transactions et des données utiles pour l'utilisateur
00000-suiviresume.json Accumule la trace des résumés successifs
00000-suivi31jours.json Garde les gains et pertes récentes
00000-resume2dates.json Garde le résumé de la transaction précédente et de l'ancienneté
00000-resume.json Garde le résumé de la dernière transaction
00000-mestransactions.json Accumule la trace des transactions
00000-mesopportunites.json Accumule la trace des opportunités de transaction
00000-gain365jours.json Garde les revenus sur 1 an pour déterminer le flux. 00000-demandeaqui.json Liste des destinataires acceptant les propositions de l'utilisateur et des proposeurs des transactions acceptées par l'utilisateur
