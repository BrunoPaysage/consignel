00000-mestransactions.json : Enregistrement des transactions de l'utilsisateur

Références/consignel-base/00/suivi transactions

Le script php écrit ce fichier pour chaque proposition ou suivi de transaction de l'utilisateur. Il le lit sur demande de l'utilisateur autorisé.

Le nom du fichier se lit
00000-mestransactions.json : identifiant de l'utilisateur
00000mestransactions.json : identifiant de la nature du fichier
00000-mestransactions.json : extension du fichier simplifiant la protection .htaccess.
Le contenu est un fichier json crypté.

Le fichier contient:
encode pour stokage {transaction}
encode pour stokage {transaction}
etc.

La transaction ou son suivi sont notés au complet comme dans le suivi horaire des transactions mais rassemblées sans répertoire horaire
