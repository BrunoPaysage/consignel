00000-resume.json : Résumé du compte

Références/consignel-base/00/resume

Le script php met à jour ce fichier pour chaque proposition et acceptation de transaction. Il le lit pour donner l'indicateur de développement durable du compte.

Le nom du fichier se lit
00000-resume.json : identifiant de l'utilisateur
00000-resume.json : identifiant de la nature du fichier
00000-resume.json : extension du fichier simplifiant la protection .htaccess.
Le contenu est un fichier json incomplet crypté.

Le fichier contient:
encode pour stokage 1,2,3,4

Les éléments de l'enregistrements sont les suivants:

  • 1 : Solde du compte en ↺
  • 2 : Minimum du compte dans les 31 derniers jours en ↺
  • 3 : Dépense de ↺ autorisée par jour
  • 4 : Maximum du compte dans les 31 derniers jours en ↺
