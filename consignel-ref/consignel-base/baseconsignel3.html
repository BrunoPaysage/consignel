Références/consignel-base/0/contrôle utilisateurs

Le script php utilise ce fichier pour former les codes de session

Contient des codes de vérification et des informations réputées publiques de l'utilisateur
1,2,A,B,C
1,2,A,B,C
etc.

1 : code chiffré de l'utilisateur code chiffré de l'utilisateur pour l'accès à la base
2 : code chiffré de de vérification de l'utilisateur
A : Nom public ou pseudonyme de l'utilisateur
B : Nom de fichier de l'image publique ou de l'avatar de l'utilisateur.
C : Localité de l'utilisateur
