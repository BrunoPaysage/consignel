00000-mesopportunites.json : Listes des transaction adressées personnellement

Références/consignel-base/00/mesopportunites

Le script php écrit ce fichier pour chaque proposition adressée personnellement. Il le lit pour fournir à l'utilisateur une liste de transaction qui lui sont adressées.

Le nom du fichier se lit
00000-mesopportunites.json : identifiant de l'utilisateur
00000mesopportunites.json : identifiant de la nature du fichier
00000-mesopportunites.json : extension du fichier simplifiant la protection .htaccess.
Le contenu est un fichier json crypté.

Le fichier contient:
encode pour stokage ["transaction1","transaction2","transaction3"]

Les éléments des enregistrements sont les suivants:

  • "transaction1" : "Transaction personnelle en attente d'approbation"
  • "transaction2" : "Transaction personnelle en attente d'approbation"
  • "transaction3" : "Transaction personnelle en attente d'approbation"
