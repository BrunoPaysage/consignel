Règles essentielles pour l'ouverture d'un compte

  • L'utilisateur est une personne physique, pas une entreprise ni un robot
  • Un seul compte par personne et une seule personne par compte
  • Il faut habiter la localité du ↺onsignél
  • Il faut être en accord avec les valeurs fondamentales du ↺onsignél

3 noms ou phrases personnelles

Vous pouvez utiliser des lettres, des nombres, des espaces et autres signes. Restez simple. L'accès au compte n'accepte aucune erreur sur un caractère, une majuscule ou un espace.

Vous devez préparer ces 3 éléments:

  • Le nom public
  • L'identifiant de compte privé
  • Le mot de passe privé

Le nom public est celui qui sera utilisé par les autres utilisateurs pour vous adresser leur proposition. Une proposition qui vous est adressée ne peut être acceptée que par vous. Vous choisissez ce nom. Il est public et unique dans la communauté locale du ↺onsignél. Donc préparez vous à ajouter un petit quelque chose pour vous distinguer si le nom est déjà utilisé.

L'identifiant de compte est la première clé d'accès à votre compte. Il est privé et unique dans la communauté locale du ↺onsignél. Donc préparez vous à ajouter un petit quelque chose pour vous distinguer si le nom est déjà utilisé. Vous seul le connaissez. Il n'est pas stocké sur le système informatique. Il ne pourra jamais être changé sans annuler totalement le compte. Vous devez donc le retenir.

Le mot de passe est la deuxième clé de sécurité pour l'ouverture de votre compte et l'accès à vos données. Il est privé. Vous seul le connaissez. Il n'est pas stocké sur le système informatique. Vous devez donc le retenir. Il pourra être changé.

Donnée de validation de personne unique

La vérification d'utilisateur unique est faite sur des identificateurs uniques préexistants. Elle est complétée par la vérification de caractéristiques individuelles vérifiables par tous. Les éléments d'identification sont agglomérés et codés ensemble pour permettre la vérification d'identité unique mais pas la recherche de la personne.

Les identificateurs uniques préexistants et les caractéristiques individuelles sont vérifié par les vérificateurs lors d'un entretien en personne avec le nouvel utilisateur. Aucune de ces données n'est enregistré sur le serveur informatique. Seule les clés de vérification sont enregistrées.

Procédure d'ouverture du compte

Le compte est ouvert en présence du nouvel utilisateur et d'un vérificateur.

  1. Le nouvel utilisateur donne au vérificateur son nom public
  2. Le vérificateur procède aux vérifications
    • de connaissance et d'accord avec les valeurs fondamentales du ↺onsignél
    • d'utilisateur unique selon des documents préexistants
    • d'habitant local
    • de caractéristiques individuelles
  3. Le vérificateur demande l'ouverture du compte du nouvel utilisateur
    1. Il ouvre son compte
    2. Il offre: vérification d'utilisateur unique
    3. Il demande: inscription d'utilisateur unique
    4. Il indique à qui avec le nom public
    5. Il indique le code de vérification
  4. Le nouvel utilisateur lit le code QR ou indique le numéro de la proposition sur le site local du ↺onsignél pour accéder à son compte. Le nom public sert de nom d'utilisateur et de mot de passe temporaire
  5. Le nouvel utilisateur change son mot de passe et entre son identifiant de compte privé
  6. Le nouvel utilisateur fait une proposition avec offre et demande au vérificateur
