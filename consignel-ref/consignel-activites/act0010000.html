act0010000 : Codage des activités

Références/consignel-activites/Codage des activités

Toutes les activités sont codées par un identifiant représentant son contenu aussi bien pour le côté client javascript que pour le çoté serveur php. Associé à la date et à l'utilisateur, cet identifiant devient une référence unique dans une transaction.

Comment interpréter l'identifiant de l'activité
act0010000 fait référence à l'activité indépendamment du fait que ce soit placé dans une offre ou dans une demande
act0010000 trois chiffres sont réservés pour identifier des activités identiques dans une même proposition. typiquement 001, 002, etc
act0010000 codage de l'activité unitaire à partir de sa description et de son unité pour une quantité. Le nombre de chiffres est variable

Ce codage des activités se retrouve dans:

  • Les offres et les demandes
  • Dans le code d'une offre particulière avec la date de transaction
  • Dans la liste du groupe des offres ou du groupe des demandes d'une transaction

Ce codage est issu de l'identification générale des activités utilisé dans les valeurs de référence.

  • Les offres et les demandes
  • Dans le code d'une offre particulière avec la date de transaction
  • Dans la liste du groupe des offres ou du groupe des demandes d'une transaction
