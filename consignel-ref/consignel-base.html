consignel-base : Stocke les données des utilisateurs.

Références/base de données

Le script serveur ↺onsignel.php vérifie l'identification des utilisateurs avant de renvoyer des données contenues dans la base.

La base contient 3 parties:

  • Pour l'identification des utilisateurs et la gestion des session consignel-base/0
  • Pour le suivi des transactions consignel-base/2
  • Pour les données des utilisateurs consignel-base/00 ou autre répertoire à plusieurs chiffres

Les différents éléments de la base sont stockés sous forme de fichiers protégés par .htaccess et cryptage
