consignel-utilisateurs-test : 2 utilisateurs pour tester

Références/consignel-utilisateurs/ consignel-utilisateurs-test.html

2 utilisateurs inscrits par défaut pour tester l’application sans configuration particulière.

Accès par menu Insc de la page d’accueil du ↺onsignél

  1. Charger la page de l'application
  2. Cliquez sur le menu "Insc"
    1. Cliquer le bouton moi ou le bouton autre selon l'utilisateur avec lequel vous voulez faire le test. Cela inscrit le nom de l'utilisateur et passe à la demande du mot de passe
    2. Cliquez sur le bouton test. Cela inscrit test dans le code secret et valide l'entrée
