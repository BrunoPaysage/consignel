Résumé

Le but du ↺onsignél est de compenser systématiquement les impacts et d'avoir les moyens de le faire.

Votre but est de viser la cible au centre.

Pas assez <-- --> Trop

Vous avez le contrôle
Si vous perdez de l’environnement ou de la qualité de vie, le rond va vers la gauche
Si vous gagnez de l’environnement ou de la qualité de vie, le rond va vers la droite

Dans l'application:
1) Vous cliquez sur la recherche d'article. Vous inscrivez ce que vous voulez offrir. Vous choisissez dans la liste ou bien vous faites un clic sur ok. Vous pouvez modifier la quantité, le titre, etc.
2) Vous cliquez sur je demande. Vous inscrivez ce que vous demandez dans la recherche d'article. Vous choisissez dans la liste ou bien vous faites un clic sur ok. Vous pouvez modifier la quantité, le titre, etc.
3) Vous cliquez sur le code QR
4) Vous vous identifiez si ce n'est pas déjà fait et vous recliquez sur le code QR après
5) Vous présentez le code QR à l'autre personne. Elle la scanne. Elle valide. C'est fait.
